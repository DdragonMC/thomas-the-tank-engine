# thomas-the-tank-engine
Thomas the Tank engine mod for Derail Valley

This mod requires Custom Car Loader to work.

Adds two variants of Thomas: one with a face and one without.
The face version has moveable eyes.
Both are unlockable as CCL worktrains for $19450 and summonable for $2500.
Both require the S060 license to operate.

Report any bugs to the #mods-support-and-bugs channel in the Altfuture Discord server.
